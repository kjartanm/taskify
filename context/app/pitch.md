# Taskify

## Description
Taskify is a web app for effective taskmanagement for kids between 10-15. Sort of Junior Trello. It should be easy, userfriendly and attractive, but still be flexible for handling typical school activities, sports, hobbies and daily routine for someone young.

## Main features
- A childres account can only be created by a parent
- A parent can manage accounts for more than one children
- A kid can have several lists of tasks for different purposes
- Tasks can have deadlines (date and time) with notifications
- You can switch between list view and kanban view
- Tasklists can be turned into repeatable checklist, with configurably frequency
- Task completion is gamified
- A parent can award badges for a certain stretch of repeated task of certain type, for example after successfully washing dishes for a month, they get a Dishwasher badge
- A parent can award taskpoints for different tasks, and set a pledge for how a certain number of taskpoints can be exchanged for a gift, a trip, or something else that the kid can look forward to
