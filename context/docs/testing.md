# Testing Documentation

This document provides comprehensive information about the testing strategy and setup for the Taskify MVP.

## Test Reports

This directory contains test reports generated by various testing frameworks.

### Generated Reports

- `cucumber-report.html` - Cucumber E2E test reports (development environment)
- `cucumber-staging-report.html` - Cucumber E2E test reports (staging environment)
- `coverage/` - Code coverage reports from Vitest
- `vitest-report.html` - Vitest test reports

### Report Generation

Reports are automatically generated when running tests:

```bash
# Generate all reports
npm run test:all

# Generate coverage reports
npm run test:coverage

# Generate E2E reports
npm run test:e2e:headless

# Generate staging E2E reports
npm run test:e2e:staging:headless
```

### Viewing Reports

- Open HTML reports in your browser
- Coverage reports are available in the `coverage/` subdirectory
- Reports are ignored by Git (see `.gitignore`)

### CI/CD Integration

In CI/CD pipelines, these reports are:
- Generated automatically
- Uploaded as artifacts
- Used for test result visualization
- Integrated with GitHub Actions for PR comments
